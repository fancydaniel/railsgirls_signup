# Rails Girls Signup

* Ruby version: ruby '2.3.0'

* Database Postgres

* To run the test suite: bundle exec rspec

* Services (job queues, cache servers, search engines, etc.)

* Deployment instructions
$ `git clone git@github.com:jendiamond/railsgirls_signup.git`  
$ `bundle`  
$ `rake db:create`  
$ `rake db:migrate`
